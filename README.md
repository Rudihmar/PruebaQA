# Prueba Automatizacion
***
Prueba automatizada realizada a la pagina https://demoqa.com/webtables , el flujo es:
1.  Ingresar al Link
2.  Agregar  Usuario
3.  Editar Usuario
4.  Eliminar Usuario

***
